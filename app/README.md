# Databricks App

Three-view Databricks App (one app, three views):

1. **Chat** — the supervisor agent, with citation chips on grounded answers and extraction-confirmation card
2. **Today** — dose checklist (mark taken/skipped), next-dose countdown, refill warnings
3. **Dashboard** — adherence % by drug, calendar heatmap, streak; caregiver mode embeds Genie space

Built from the chat app template (Streamlit or React), deployed as a URL for judges.

Phase 3 (state + app) and Phase 5 (polish) deliverable. See ARCHITECTURE.md §7.
